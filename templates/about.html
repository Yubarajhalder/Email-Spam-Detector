About the Project

A machine learning-powered email spam detection system

Project Overview

The AI Email Spam Detector is a web application that classifies email messages as Spam or Ham (legitimate) using a supervised machine learning pipeline. It combines natural language processing with a trained classifier to deliver fast, accurate predictions through an intuitive web interface.

The system was built with Python Flask on the backend, Bootstrap 5 on the frontend, and Scikit-learn for the machine learning pipeline — making it lightweight, portable, and easy to deploy.

What is Spam?

Spam refers to unsolicited bulk messages — often promotional, fraudulent, or malicious. Common examples include phishing attempts, lottery scams, fake prize notifications, and unwanted marketing emails. Spam wastes time, consumes bandwidth, and can pose serious security risks.

What is Ham?

Ham is the term used in email filtering for legitimate, wanted messages. These include personal correspondence, work emails, appointment reminders, and messages from trusted senders. The goal of spam detection is to let ham through while blocking spam.

Text Input

User pastes email content into the web form.

TF-IDF Vectorization

Text is converted to a numerical feature vector using TF-IDF weighting.

Classification

Logistic Regression predicts Spam or Ham with confidence scores.
